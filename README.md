# Паркет36 — мастер Иван / parket36.ru

Статический сайт для заявок на паркетные работы, деревянные полы, мелкий ремонт и помощь профильных мастеров в Воронеже и Воронежской области.

## Основное позиционирование

- Паркет, циклёвка, реставрация, деревянные полы и укладка покрытий — основная специализация.
- Иван лично принимает обращения и объясняет формат выполнения.
- Сложная электрика, сантехника, отделка, демонтаж, перевозки и другие профильные задачи передаются отдельным специалистам только после согласования с клиентом.

## Автоматическая проверка

Каждый Pull Request и каждое изменение ветки `main` проверяются workflow `Site quality`.

Проверяются:

- внутренние ссылки и изображения;
- title, description, canonical и H1;
- наличие страниц в sitemap;
- отсутствие WhatsApp и устаревших ссылок;
- отсутствие видимых SEO-списков;
- корректный `noindex` для 404, политики и страницы-переходника;
- отсутствие ссылок на объединённую страницу `master-na-chas`.

Локальный запуск:

```bash
python tools/check_site.py
```

## Публикация

Workflow `Deploy GitHub Pages` после изменения `main`:

1. запускает аудит;
2. собирает чистую публичную папку `_site`;
3. загружает только файлы сайта;
4. публикует артефакт в GitHub Pages.

Локальная сборка:

```bash
python tools/build_pages.py
```

Для работы автоматического деплоя в `Settings → Pages` источником должен быть выбран `GitHub Actions`.

## Домен

Файл `CNAME` содержит:

```text
parket36.ru
```

DNS корневого домена должен быть направлен на GitHub Pages. Изменения DNS выполняются в кабинете регистратора домена.

## MAX

Пока кнопки ведут на общую страницу:

```text
https://max.ru/
```

После получения персональной ссылки Ивана нужно заменить её во всех HTML-файлах. Автоматический аудит оставляет предупреждение, пока используется общая ссылка.

## Реальные фотографии и кейсы

В портфолио пока используются тематические иллюстрации. Для каждого нового объекта желательно сохранять:

1. общий вид до начала работ;
2. дефект крупным планом;
3. процесс;
4. общий вид результата;
5. площадь, район, срок и состав работ.

После получения материалов фотографии обрабатываются, оптимизируются и оформляются как отдельный кейс.
