# Паркет36 — мастер Иван / parket36.ru

Сайт для сбора заявок на муж на час, мастер на час, мелкий ремонт, отделку, электрику, сантехнику, демонтаж, вывоз мусора, переезды, сборку мебели, паркет и полы.

## Как обновить сайт на GitHub

1. Распакуйте архив.
2. Откройте репозиторий `parket36-site`.
3. Удалите старые файлы или загрузите новые поверх старых через `Add file` → `Upload files`.
4. Перетащите все файлы и папки из распакованного архива.
5. Нажмите `Commit changes`.
6. В `Settings` → `Pages` должно быть: `Deploy from a branch`, ветка `main`, папка `/ (root)`.

## Как подключить MAX

Сейчас кнопки MAX ведут на `https://max.ru/`. Когда Иван получит свою персональную ссылку в MAX, замените во всех HTML-файлах:

`https://max.ru/`

на ссылку вида:

`https://max.ru/u/ВАШ_КОД`

Быстро заменить можно поиском по репозиторию: найдите `https://max.ru/` и замените на персональную ссылку.

## Как поменять телефон

Найдите во всех файлах:

`8 (900) 926-79-29` и `+79009267929`

и замените на новый номер.

## Как заменить фото

1. Загрузите фото в папку `/img/`.
2. Откройте `/portfolio/index.html`.
3. Замените пути вида `/img/work-repair.svg` на `/img/rabota-1.jpg`.
4. В `alt` укажите осмысленное описание услуги и город.
