# Паркет36 — готовый сайт для GitHub Pages

Готовый статический сайт для домена parket36.ru.

## Быстрое обновление
1. Откройте репозиторий на GitHub.
2. Нажмите Add file → Upload files.
3. Перетащите новые файлы из этой папки.
4. Нажмите Commit changes.
5. GitHub Pages обновит сайт автоматически.

## Как заменить фото
1. Положите фотографии в папку img.
2. Откройте portfolio/index.html.
3. Замените пути вроде /img/before-after.svg на ваши: /img/rabota-1.jpg.
4. Поменяйте alt, например: Циклёвка паркета в Воронеже — до и после.
