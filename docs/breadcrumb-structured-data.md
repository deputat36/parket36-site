# Структурированные хлебные крошки

Публичная сборка автоматически создаёт `BreadcrumbList` JSON-LD из видимой навигации `.breadcrumbs`.

## Зачем

На страницах услуг, решений и советов уже есть понятные хлебные крошки для посетителя. Генератор передаёт ту же иерархию поисковикам без ручного дублирования разметки в десятках HTML-файлов.

## Правила генерации

- обрабатываются только страницы с видимым блоком `.breadcrumbs`;
- страницы с `noindex` пропускаются;
- корневая страница без хлебных крошек не получает искусственную цепочку;
- последний элемент всегда связан с canonical страницы;
- все URL должны принадлежать `https://parket36.ru`;
- разделители `›`, `>`, `→`, `/` и `»` не попадают в названия элементов;
- повторная генерация не создаёт второй блок;
- исходные HTML-файлы не изменяются — разметка добавляется только в `_site`.

## Проверка

Самопроверка генератора:

```bash
python tools/check_breadcrumb_schema.py
```

Полная публичная сборка:

```bash
python tools/build_pages.py
```

Обе команды входят в общий quality gate:

```bash
python tools/run_quality_checks.py
```

Генератор находится в `tools/breadcrumb_schema.py`. В готовом HTML блок отмечен атрибутом `data-generated-breadcrumbs`.
