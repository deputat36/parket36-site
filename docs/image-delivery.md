# Изображения в публичной сборке

## Цель

Публичные страницы должны заранее сообщать браузеру реальные размеры изображений. Это уменьшает сдвиги макета при загрузке и не позволяет случайно публиковать изображения без текстового описания.

## Как работает сборка

`tools/image_attributes.py` обрабатывает все теги `<img>` после копирования сайта в `_site`.

Для локального изображения сборка:

1. проверяет наличие `src` и `alt`;
2. убеждается, что файл существует;
3. определяет реальные размеры SVG, PNG, JPEG или другого формата, который читает Pillow;
4. записывает фактические `width` и `height` в HTML;
5. добавляет `decoding="async"`;
6. добавляет `loading="lazy"` только временным изображениям с `data-placeholder-image` или именем `work-*`.

Если исходный HTML содержит неправильные размеры, публичная сборка заменяет их значениями реального файла.

## Правила для новых фотографий

- Использовать собственные подтверждённые фотографии Ивана и объектов.
- Всегда писать содержательный `alt`, описывающий то, что важно клиенту на изображении.
- Не оставлять пустой `alt`, если изображение не является декоративным.
- Для декоративного изображения использовать `alt=""` вместе с `aria-hidden="true"` или `role="presentation"`.
- Не ставить `loading="lazy"` на главное изображение первого экрана.
- Для изображения первого экрана при необходимости использовать `fetchpriority="high"`.
- После замены файла запускать полный quality gate, чтобы размеры пересчитались и проверились.

## Проверки

```bash
python tools/check_image_attributes.py
python tools/build_pages.py
```

Первая команда выполняет изолированную самопроверку нормализатора. Вторая строит `_site` и проверяет все реальные изображения проекта.

## Что блокирует публикацию

- отсутствующий или пустой `src`;
- отсутствующий `alt`;
- пустой `alt` без признака декоративного изображения;
- отсутствующий локальный файл;
- SVG без определяемых размеров или `viewBox`;
- растровый файл, который Pillow не может прочитать;
- отсутствие изображений в публичной сборке, указывающее на поломку шага.
