# Отчёт по карточкам раздела советов

Инструмент `tools/report_advice_index_gaps.py` помогает быстро проверить связь между `sitemap.xml` и каталогом `/sovety/`.

Запуск:

```bash
python tools/report_advice_index_gaps.py
```

Что проверяется:

- страницы `/sovety/.../`, которые есть в `sitemap.xml`, но не выведены карточкой в `/sovety/`;
- карточки в `/sovety/`, которые ведут на страницу советов, но не имеют записи в `sitemap.xml`;
- общее количество страниц советов в sitemap и карточек в каталоге.

Когда запускать:

- после добавления новой статьи в `/sovety/`;
- после добавления или удаления карточки в `sovety/index.html`;
- перед Pull Request, если менялись статьи, sitemap или каталог советов.

Нормальный результат — сообщение, что sitemap и карточки `/sovety/` синхронизированы.
